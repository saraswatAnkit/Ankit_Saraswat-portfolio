Optimizing Feedback Notification and Response History

By Ankit Saraswat | January 2, 2025

Introduction

In feedback systems, managing notifications and response history effectively is critical for ensuring transparency and flexibility. This post outlines how I tackled a challenge involving maintaining admin responses and giving users control over their visibility.

The Challenge

The requirements were twofold:

  • Maintain a history of all admin responses to ensure accountability.
  • Allow users to control whether a response is visible to other users.

The Solution

To address these requirements, I implemented the following:

  1. Database Design for History Maintenance: A dedicated table to store admin responses.
  2. Conditional Display Logic: APIs allowed users to toggle visibility of the responses.
  3. Threaded Chat Integration: Notifications were triggered only when the user opted for visibility.

Outcome

This solution resulted in:

  • A robust history management system for admin responses.
  • Empowered users with visibility control for feedback responses.
  • Context-aware notifications integrated into threaded chats.
